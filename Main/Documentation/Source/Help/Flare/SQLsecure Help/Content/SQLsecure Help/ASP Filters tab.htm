Change audit filters

The Filters tab allows you to add a new filter, delete filters, and edit filter properties. Filters are the criteria used when collecting audit data for your snapshots.

Option

Description

New Filter

This button opens the Add Filter Wizard, which allows you to add filters to your snapshot. For more information, see Using the Add Filter Wizard.

Delete Filter

Click this button to delete the selected filter.

Properties

This button opens the Filter Properties dialog, which allows you edit the selected filter settings. For more information, see Edit Filter Properties.
How do I add a new audit filter?

When you click Add New Filter, the New Filter wizard opens. To add a new filter, follow the directions on the windows in the wizard. For more information, see “New Filter wizard”.

 How do I edit an audit filter?

Select the filter you want to edit and click Edit to show all the properties of the audit filter.

How do I delete an audit filter?

Select the filter you want to delete and click Delete to remove the filter from SQLsecure.

What do I do next?

Once you have configured your snapshot collection settings, you can view your updated snapshots to ensure that they are set up the way you intended.

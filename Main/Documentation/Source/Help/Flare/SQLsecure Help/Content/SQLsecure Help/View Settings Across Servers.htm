View settings across all servers

The Settings tab lists the security settings collected for all SQL Server instances associated with this policy. By default, SQLsecure sorts this data by setting name. To sort by instance name, click By Server in the grid menu bar.

You can view a more detailed list of settings, and edit the way SQLsecure collects data from a SQL Server instance, by double-clicking the SQL Server instance from the By Server list.

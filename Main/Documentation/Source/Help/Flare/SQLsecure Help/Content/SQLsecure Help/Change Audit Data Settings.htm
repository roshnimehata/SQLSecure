Change Audit Data Settings

The Change Settings button opens the Select Audit Data window. This window allows you to select what data to display in your Security Report Card. By default, Use most current data is selected. To select alternate data, uncheck the Use most current data box and select the date and time of the data you want displayed in the Security Report Card.

Select the Use baseline snapshot data only to populate the Security Report Card with snapshots that have been marked as a baseline.

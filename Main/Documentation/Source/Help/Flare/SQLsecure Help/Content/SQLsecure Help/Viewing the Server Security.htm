View Server Report Card

The Server Security Report Card lists the security checks evaluated for the selected SQL Server instance.

How are the risks organized on the Server Security Report Card?

The default view of the Security Report Card displays all the associated security findings, from highest to lowest risk, as configured in your policy. You can select security risk categories along the left side of the report card to filter the report card accordingly.

 What is the Server Status?

The Server Status section lists the number of security check violations found along with the level of risk associated with these findings. This status reflects the findings for the selected instance only.

 How can I get more information on discovered risks?

The following areas of SQLsecure can provide detailed information on the associated risks:

Settings

The Settings tab of the Security Summary lists the detailed SQL Server settings associated with the selected SQL Server instance.

Users

The Users tab of the Security Summary lists the logins associated with the policy along with account information.

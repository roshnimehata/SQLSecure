Configure SQLsecure

You must first register SQL Server instances with SQLsecure, setup your audit filters, and schedule your audits before SQLsecure you can begin auditing your SQL Server instances.

To successfully audit your SQL Server instances, perform the following steps:

  1. Upgrade to your permanent product license
  2. Add a SQL Server to SQLsecure
  3. Add a Snapshot Filter
  4. Schedule your Snapshots

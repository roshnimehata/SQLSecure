Specify login properties

The Add a new SQL Server login window allows you to enter the Windows account name of the user you would like to add to SQLsecure. You can also specify whether or not you would like to grant access to this user.

By selecting the Deny access option, the user will be blocked from accessing SQLsecure.

TIP

 The user account must be entered in a domain\username and case-sensitive format.

Edit login settings

SQLsecure allows you to edit individual login permissions (to SQLsecure) from the Manage SQLsecure view.

To edit SQLsecure login properties:

  1. Select Logins from the Manage SQLsecure view.
  2. Right-click the login you want to edit and select Properties.
  3. Edit the SQLsecure access properties and click OK to queue your changes. The SQLsecure login has been edited and the new settings will apply the next time the user logs into SQLsecure.

Select policy template

The Select the Policy Template window allows you to choose to create a new policy "from scratch" or base your policy on a built-in template.

What are policy templates?

Policy templates are boilerplate policies whose security check definitions reflect known industry regulations and security best-practices. Although you may choose to start with a template, you can later add, edit, or remove security checks as needed. For more information, see how policy templates can help you achieve your SQL Server security goals.

 Can I create my own templates?

Yes. You can create policy templates by exporting configuration settings from a specific policy to an XML file. Then, you can later reuse these settings by creating new policies based on this template. For more information, see Import Policy.

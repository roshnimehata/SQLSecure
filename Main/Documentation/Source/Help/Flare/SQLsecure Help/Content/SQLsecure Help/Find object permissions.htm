Find Object Permissions

SQL Server database objects can be viewed in the Explore Permissions view.

To find all the permissions associated with a particular user:

  1. Select the SQL Server instance you would like to search from the Audited SQL Servers list.
  2. Select the Object Permissions tab.
  3. Use the Snapshot tree to view object properties.
  4. Click Save to save your results or Print to print out the properties of the object.

For more information on permissions, see Permissions in SQLsecure.

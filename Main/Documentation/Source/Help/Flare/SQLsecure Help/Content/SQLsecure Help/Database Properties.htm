View database properties

The Database Properties window displays the permissions information associated with the selected database. You can view:

  • The owner and status of the selected database
  • Whether the guest SQL Server login is enabled on this database
  • The account or login (grantee) that was granted or denied permissions on the database
  • The type of permission, and whether it was granted or denied
  • The account or login (grantor) granting or denying this permission
  • The source permission, object, and type from which the effective permission was inherited
What can I do on the Database Properties window?

You can view explicit permissions only, or include the fixed role and inherited permissions, by checking the appropriate option and clicking Show Permissions. You can also save or print the database object permissions information by clicking the appropriate icon above the permissions table.

Manage your license

SQLsecure provides an intuitive, simple to use interface for license key management. You can view the status of the license keys and add licenses to audit additional instances.

When you reach the SQL Server limit dictated by your license, SQLsecure will not let you add new servers. If your trial period expires, you will be prompted for a new license when you start the SQLsecure Console. If a valid license is not provided, then SQLsecure will shut down and no longer be accessible.

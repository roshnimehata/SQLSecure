Add new filter

The Add Filter wizard allows you to choose the appropriate criteria to use when collecting snapshot information. By default, SQLsecure collects all security information.

TIP

Consider using the default filter settings in your initial policy assessments until you know exactly which data your policies will require. Using the default filter settings ensures that all the data required by your policies is collected.
How do I access this wizard?

To open the Add Filter wizard:

  1. Right-click the server you want to edit, and select Properties.
  2. Select the Filters tab.
  3. Click New Filter.
 What does the Add Filter wizard do?

The Add Filter wizard lets you designate what types of permission and security data will be collected.

How does the Add Filter wizard work?

The Add Filter wizard stores your filters in the Repository database, which are then used by SQLsecure when taking snapshots.

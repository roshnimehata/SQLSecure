Edit audited SQL Server properties

The Audited SQL Server Properties window allows you to edit the way SQLsecure monitors your SQL Server instances. The Audited SQL Server Properties window contains general SQL Server instance information, snapshot filter settings, and audit data collection schedule.

How do I open the Audited SQL Server Properties window?

To access the Audited SQL Server Properties window, right-click the SQL Server instance from the Explore Permissions view and select Configure Audit Settings.

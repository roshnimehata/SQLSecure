Select SQL Server objects to audit

The Select SQL Server Objects to Audit window allows you to specify the objects about which SQLsecure should collect security information. By default, SQLsecure audits all SQL Server objects.

TIP

When you are selecting objects to audit, be aware that you need to include all the objects that your policies need to appropriately assess security risks.
How do I select objects to audit?
  1. Check the objects in the list.
  2. For those objects that have scope options, click the blue text in the Scope matches column and select the appropriate option.
  3. For those objects that have naming options, click the blue text in the Name matches section, and enter the appropriate matching convention.

How to use the Deploy Reports wizard

You can deploy the SQLsecure Reports to your existing Microsoft Reporting Services installation.

If you previously deployed SQLsecure Reports, verify which version of Reporting Services is currently running in your environment. SQLsecure supports Reporting Services version 2005 or later.

TIP

If you are upgrading reports from SQLsecure 2.0, delete all of the previously installed SQLsecure reports before deploying the new reports.

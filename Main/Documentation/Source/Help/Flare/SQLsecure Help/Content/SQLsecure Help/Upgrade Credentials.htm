Upgrade Credentials

SQLsecure no longer uses the credentials of your SQL Server Agent to collect Operating System and SQL Server security information. This change improves the security model of SQLsecure. If, in a previous version of SQLsecure, it was specified that SQLsecure should use the credentials of the SQL Agent to collect security information, you will have to enter new credentials.

The credentials you enter on the Upgrade Credentials window are applied to all registered SQL Server instances (where you had previously elected to use the default SQL Agent permissions to collect data). If you cannot use the same credentials for all your registered SQL Server instances,you will need to update the credentials used on the SQL Server Properties Window for each of your SQL Server instances. For more information, see Credentials.

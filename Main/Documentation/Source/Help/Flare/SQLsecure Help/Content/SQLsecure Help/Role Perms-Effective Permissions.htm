Effective role permissions

The Effective Permissions tab lists all the effective permissions the selected role has on objects in the target database.

Effective permissions are the net effect of assigned permissions and permissions inherited from any parent roles.

Be aware that calculating effective permissions can take several minutes to run.

What information is on the Effective Permissions tab?

The Effective Permissions tab contains server and database permission information for the selected role. This includes the object names and types, the type of access granted to the role members, and who granted these permissions.

 What do I do on the Effective Permissions tab?

Click Calculate Effective Permissions to view all the effective permissions the selected role has (at the time the data collection was taken) on the SQL Server instance being audited.

 What do I do next?

You can save the permissions information to an Excel spreadsheet, print the permissions search, or select a different user to search for.

To save or print the permissions information, right-click the grid and choose the appropriate option.

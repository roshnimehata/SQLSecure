Configure email settings

The SMTP Email Provider window allows you to enter your provider information, through which email notifications are distributed.

To configure your SMTP Email Provider:

  1. Enter the SMTP address, port number, and the number of seconds to wait before timing out.
  2. If the logon for the SMTP Email Provider requires authentication, check the Server requires authentication box and enter the user name and password.
  3. Enter the name and address in the Sender Information section that will appear in the From field on the email notifications that are distributed by SQLsecure.

View single server summary

The Server Summary window allows you to view the following:

  • General SQL Server properties
  • Auditing status of the SQL Server instance
  • Audit history
What kind of information is contained in SQL Server Properties?

SQL Server Properties displays the name of the instance, the version of SQL Server being used, and the Windows Operating System the SQL Server instance is operating on.

What kind of information is contained in Auditing Status?

Auditing Status displays the status of the last collection, the date and time of the next scheduled collection, and basic statistics for the latest snapshot.

What kind of information is contained in Audit History

Audit History provides detailed information about your snapshot, including the time it was taken, the collection statistics, audit filter information, and a listing of any suspect Windows accounts or unavailable databases that were encountered while the Snapshot was being taken. For more information on suspect Windows accounts, see Identify Suspect Windows Accounts. For more information on unavailable databases, see Identify Unavailable Databases.

The following table describes the information displayed in each of the columns:

Item Description

Date/Time

The date and time when the snapshot was taken

Status

The status of the snapshot (audit data collection)

Comments

Description of any issues the collector encountered

Baseline

Whether or not the snapshot is marked as a baseline

Objects

The number of objects audited in the snapshot

Permissions

The number of permissions collected in the snapshot

Logins

The number of logins collected in the snapshot

Group Members

The number of group members collected in the snapshot

Import policies

The Import Policy window allows you to choose to either add a policy template or previously exported policy to the Policy tree. Once the policy is selected, you can then configure the policy and select the SQL Server instances to add to the policy.

How do I import a policy?
  1. If you want to import from a policy template, select the Import from policy template box, select the template to import, and then click OK. For more information, see how policy templates can help you achieve your SQL Server security goals.

    2. If you want to import a previously exported template, select the Import previously exported policy box and click Browse to locate the policy template.

  2. Select each of the tabs on the Import Policy windows and perform the following actions to configure the policy.

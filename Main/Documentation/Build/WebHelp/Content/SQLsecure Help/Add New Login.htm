Add new login

The New Login Wizard allows you to add new SQL Server logins to SQLsecure. In this wizard, you will be asked to supply the Windows user account name and level of access for the user you would like to add.

To access the SQLsecure New Login wizard, click New Login at the top of the Logins window in Manage SQLsecure.

TIP

A Repository login (database user) can be assigned SQLsecure permissions. These permissions control what tasks you are able to do in SQLsecure.

To add a user login to SQLsecure: 

  1. Select File > New SQLsecure Login from the toolbar.
  2. Enter the Windows user account name to add on the Add a new SQL Server login window.
  3. Select the Grant access option and click Next.
  4. Select the type of access to grant the user (either administrative or viewing privileges) and click Next.
  5. Click Finish to add the login to SQLsecure.

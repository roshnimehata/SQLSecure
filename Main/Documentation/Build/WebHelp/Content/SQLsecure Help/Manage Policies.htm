Manage policies

Use the Manage Policies view to obtain a quick overview of which policies are enabled and have saved assessments.

Columns

Assessments

Indicates how many assessments have been saved from this policy.

Date

Indicates the date and time of the audit data used by this assessment. SQLsecure pulls the audit data from the snapshot that most closely matches this date and time.

Description

Provides the description specified for this policy or assessment.

Dynamic

Indicates whether this policy dynamically includes any newly registered SQL Server instance.

High Configured

Indicates how many high risk security checks were configured for this policy or assessment.

High Icon

Indicates how many high risk findings this policy or assessment had when last run.

Medium Configured

Indicates how many medium risk security checks were configured for this policy or assessment.

Medium Icon

Indicates how many medium risk findings this policy or assessment had when last run.

Name

Provides the name of the policy or assessment.

Low Configured

Indicates how many low risk security checks were configured for this policy or assessment.

Low Icon

Indicates how many low risk findings this policy or assessment had when last run.

Review Notes

Indicates whether this policy has Internal Review Notes.

State

Provides the state (Draft, Published, Approved) of this assessment.

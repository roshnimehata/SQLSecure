Choose to take snapshot now

The Take Snapshot window gives you the option to collect audit data at the end of the registration process. Choose this option when you want to immediately perform a security assessment on the newly registered instance. Note that you can also manually take a snapshot later by selecting Take Snapshot on the Explore Permissions view.

If you are adding several SQL Server instances to SQLsecure, you may not want to take a snapshot at the end of the Register SQL Server wizard process. Taking a snapshot from more than one instance at a time can degrade the Console performance.
